# littlefocus
小专注 - 极简专注,不负时光

一个轻量级的移动端专注应用，集成了专注计时、待办任务、习惯养成和数据统计功能。
---

## ✨ 功能特性

### 🎯 专注计时
- **倒计时模式**：预设时长专注（1-120分钟可自定义）
- **正计时模式**：自由专注，从0开始累加
- **圆形进度条**：实时显示剩余时间比例
- **自动记录**：倒计时自然结束时自动记录专注数据并播放提示音
- **任务/习惯关联**：从待办或习惯页面开始专注，自动关联统计

### 📋 待办任务
- 添加、编辑、删除任务
- 三级优先级（高/中/低）
- 自定义分类管理（工作、学习、生活等）
- 按分类筛选查看
- 累计专注时长/次数统计

### ❤️ 习惯养成
- 每日打卡功能
- **真实的连续天数计算**：只有昨天打卡，今天打卡才延续连续天数
- 每日自动重置打卡状态（连续天数保留）
- 累计专注时长/次数统计
- 今日完成率展示

### 📊 数据统计
- 多维度统计（今日/本周/本月/自定义范围）
- 专注次数、总时长、日均时长
- 专注类型分布（任务专注/习惯专注）
- **月专注日历**：可视化展示每日专注分钟数

### 🎵 白噪音
- 雨声、森林、篝火三种音效
- 点击切换，自动停止当前音效
- 可随时开启/关闭

### 🔆 屏幕常亮
- 开始专注时自动开启
- 暂停/停止专注时自动关闭
- 无需手动操作

### 🎨 主题切换
- 8种配色主题可选
- 默认绿、清新蓝、温柔粉、优雅紫
- 青柠绿、天空蓝、玫瑰红、夜间模式

### 💾 数据管理
- 本地存储（localStorage），关闭页面不丢失
- 数据导出（JSON格式备份）
- 数据导入（恢复备份）
- 一键清空所有数据
- 每日自动清理已完成任务

## 🚀 快速开始

### 在线访问
访问 GitHub Pages 地址即可使用：https://Wangjh228.github.io/littlefocus/

### 本地运行
1. 下载 `index.html` 文件
2. 用浏览器打开即可使用（推荐手机浏览器或 Chrome 开发者工具手机模式）

### 添加到主屏幕
- **iOS Safari**：点击分享按钮 → 添加到主屏幕
- **Android Chrome**：点击菜单 → 添加到主屏幕

> ⚠️ 注意：更建议iOS系统使用

## 🎯 使用场景

- **番茄工作法**：25分钟专注 + 5分钟休息
- **任务驱动**：为每个任务记录专注时间
- **习惯培养**：每日打卡 + 专注记录双维度追踪
- **放松辅助**：白噪音营造专注氛围
- **数据分析**：通过统计了解自己的专注规律
